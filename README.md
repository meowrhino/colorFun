# colorfun (v0)

Web estática vanilla centrada en un playground de color.

## Ejecutar
Usa servidor local (recomendado):
- VSCode Live Server
- o: `python -m http.server` y abre `http://localhost:8000/colorfun/`

## Estructura
- `data/htmlNamedColors.json` define grupos y colores.
- `index.html` carga la app principal.
- `js/colorPlayground.js` contiene estado, lógica y eventos del playground.
- `styles.css` define layout desktop/móvil.
